# Managing Documentation

Skill collection for authoring and reviewing project documentation. Works in any environment where the agent can read and write repository files.

## Overview

Creates and reviews Markdown and AI-facing documentation — skills, rules, `AGENTS.md`, and similar instructional content — for clarity, structure, style, and formatting.

## Agent workflow

Follow this skill when creating or updating Markdown docs, skills (`SKILL.md`), rules, `AGENTS.md`, or similar instructional content — or when reviewing those docs for quality before merge or publication. Match one **Recipes** row; open exactly that reference.
